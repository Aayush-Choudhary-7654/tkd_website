# 🥋 FRONTEND PRD

## ACTIVE TAEKWONDO WEBSITE

---

## 1. 📌 Overview

**Product Name:** ACTIVE TAEKWONDO Website
**Scope:** Frontend (UI/UX + Layout + Visual Design)

**Objective:**
Design a high-converting, modern, and responsive homepage that:

* Attracts new students
* Builds trust
* Encourages sign-ups (Join Now / Free Trial)

---

## 2. 🎯 Design Philosophy

* Modern + Sporty UI
* Martial arts aesthetic
* Strong, bold visuals
* High contrast (Dark Theme)

### 🎨 Color Palette

* Background: #0A0A0A (Black)
* Primary: #FF2D2D (Red)
* Secondary: #1F1F1F
* Text: #FFFFFF

### 🔤 Typography

* Heading: Bebas Neue / Oswald (Bold, athletic)
* Body: Poppins / Inter

---
